User Stories for Scenario 1: Arthur, Family Dinner

As a customer, I want to be able to schedule reservations.

As a customer, I want to be able to make a reservation in advance.

As a user of SmartReserve, I need to be able to state how many people are in my party.

As a user, I want to say how many adults and children there are in my party so that I can get highchairs when needed.

As a customer, I need to be able to schedule reservations for a specific date and time.

As a user, I want to edit my reservations to add or take out people from my party.

As a customer, I would like to be told if the restaurant is booked for a day and time.

As a user of SmartReserve, I need the host to see my reservations under my name for a certain number of people at a specific date and time.


User Stories for Scenario 2: Arthur, Date Night

As a customer, I need to be able to schedule reservations at the last minute.

As a customer, I want to be able to edit my reservation information.

As a user, I want to book my reservation in a specific location so that I can be in a quieter and more intimate setting.

As a user of SmartReserve, I need to be able to save changes on my reservation.

As a customer, I need to be able to cancel my reservation.

As a user, I want to be able to change my reservation for a different day and time.

As a user, I want to be able to keep some information the same so that I do not have to input it twice.

As a customer, I want to be able to change only one detail of my reservation.


User Stories for Scenario 3: Arthur, Dinner with Friends

As a user, I need to see if the restaurant is booked when trying to schedule a reservation.

As a customer, I want the host to see the total occupancy in the restaurant.

As a customer, I need the host to see all reservations for the restaurant.

As a user of SmartReserve, I want the host to see how many people are in every party in the restaurant.

As a user, I need the host to be able to see changes made to the tables and reservation list.

As a customer, I need the host to see if a table is open or occupied.

As a user, I want the host to be able to edit the occupancy in the restaurant.

As a customer, I need the host to be able to edit the table to open or occupied so that tables are not double booked.
