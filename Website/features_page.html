Feature List


Description

As a user, I want to hit “Make a Reservation” to make a new reservation for any date and time.

As a user, I need to be able to click “Edit Reservation” to edit my reservation.

As a user, I need to be able to click “Cancel Reservation” in order to cancel my reservation.

As a user, I need to click “Confirm Reservation” in order to book the reservation.

As a user, I need to be able to click “Confirm Changes” to confirm the changes made to my reservation.

As a user, I want another confirmation to make sure I want to cancel my reservation so that I do not accidentally cancel a reservation.

As a user, I want the “Cancel Reservation” button at the bottom of the screen so that I don’t accidentally click it.

As a user, I need to be able to state how many people are in my party.

As a user, I want to say how many adults and children there are in my party so that I can get highchairs when needed.

As a user, I want to edit my reservations to add or take out people from my party.

As a user, I would like to be told if the restaurant is booked for a day and time.

As a user, I need the host to see my reservations under my name for a certain number of people at a specific date and time.

As a user, I want to book my reservation at either a booth or table.

As a user, I want to book my reservation by a window.

As a user, I want the option to have no preference when it comes to sitting in a booth, table, or by a window.

As a user, I need to be able to call the restaurant in order to book my reservation.

As a user, I need to be able to call the restaurant in order to edit my reservation.

As a user, I need to be able to call the restaurant in order to cancel my reservation.

As a user, I want the application to be able to update immediately when a reservation is changed.

As a user, I need the restaurant to see that a reservation was canceled.

As a user, I want the host to be able to make a reservation in the same way a customer can.

As a user, I need to be able to see what is available for multiple days and times so that I can choose which works the best for me.


As an employee, I want to see the total occupancy in the restaurant.

As an employee, I need to see all reservations for the restaurant.

As an employee, I need to see how many people are in every party in the restaurant.

As an employee, I need to be able to see changes made to the tables and reservation list.

As an employee, I need to see if a table is available, occupied, or reserved.

As an employee, I want to be able to edit the occupancy in the restaurant.

As an employee, I need to be able to edit the table to available, occupied, or reserved so that tables are not double booked.

As an employee, I need to be able to create reservations.

As an employee, I need to be able to cancel reservations.

As an employee, I need to be able to identify reservations with personal information.

As an employee, I want to be able to make new tables for the restaurant.


Constraints

The use of some tools may be limited based on restaurant costs and user abilities and costs.


Comments

Based on Jameson, Arthur, Julissa, Susan, Michael, and John’s Scenarios
